# @christophersesugh/memofs-mcp-server

## 1.0.0-alpha.0

### Major Changes

- Initial release of `@christophersesugh/memofs-mcp-server` — Model Context Protocol server for Memo FS.

 - Exposes Memo FS memory operations as MCP tools for LLM agents
 - `memofs-mcp` binary — drop-in stdio MCP server
 - Configurable runtime mode (`local`, `memory`, `cloud`, `hybrid`)
 - `@modelcontextprotocol/sdk` included as a direct dependency — no separate install required
